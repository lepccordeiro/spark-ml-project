# spark-ml-project
End-to-end PySpark pipeline for data processing and machine learning, including feature engineering, PCA, regression modeling, SQL optimization, and Parquet-based storage.


# 📊 Spark ML Pipeline - Video and Comments Analysis

This project implements a complete data processing and machine learning pipeline using PySpark, with a focus on performance optimization and scalable data analysis.

## 🚀 Features

- Data ingestion from Parquet files
- Data preprocessing and feature engineering
- Categorical variable indexing
- Feature vector creation using VectorAssembler
- Data normalization (StandardScaler)
- Dimensionality reduction with PCA
- Linear Regression model training
- Model evaluation (RMSE)
- Spark SQL queries and joins
- Query optimization (projection, filtering, joins)
- Execution plan analysis using `explain`
- Efficient storage in Parquet format

## ⚙️ Technologies

- PySpark
- Apache Spark
- Google Colab
- Python

## 📁 Project Structure

- `videos-preparados.snappy.parquet`
- `video-comments-tratados.snappy.parquet`
- Notebook containing the full pipeline

## 🎯 Objective

To demonstrate the application of data engineering and machine learning techniques in a distributed environment, focusing on performance and best practices.
